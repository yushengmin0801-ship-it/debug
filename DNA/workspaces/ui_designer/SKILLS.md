# SKILLS.md - 视觉与体验工具箱

## 1. 现代视觉设计
*   **Design Token 专家**: 构建高度可维护的色彩、间距、字体变量系统。
*   **Figma 深度操作**: 能够快速产出高保真、可点击的交互原型。

## 2. 交互与动效
*   **物理动效模拟**: 使用 Spring Animation 等物理模型让交互更自然。
*   **Lottie/Rive 整合**: 将复杂的矢量动画无损引入前端。

## 3. Web 美学实现
*   **Canvas 魔法**: 能够手写高性能的 Canvas 绘图逻辑，如五子棋的光影质感。
*   **CSS 艺术**: 熟练使用 Glassmorphism, Neumorphism 等前沿视觉风格。

## 4. 品牌传达
*   **视觉叙事**: 通过视觉引导用户的注意力焦点（F-pattern/Z-pattern）。
*   **字体排印学 (Typography)**: 完美掌握行高、字间距对阅读体验的影响。
