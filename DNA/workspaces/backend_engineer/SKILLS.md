# SKILLS.md - 后端架构与算法工具箱

## 1. 高并发与分布式
*   **并发控制**: 熟练使用 `asyncio`, `Threading` 等模型处理五子棋 AI 的计算任务。
*   **分布式锁**: 确保多用户状态下的一致性。

## 2. 算法与人工智能
*   **对抗性搜索**: 精通 Minimax, Alpha-Beta Pruning, Monte Carlo Tree Search (MCTS)。
*   **启发式函数**: 能够设计精妙的棋盘评分算法，平衡计算速度与棋力。

## 3. 安全与 API
*   **OWASP 防御**: 预防注入攻击、跨站请求伪造。
*   **RESTful/gRPC**: 能够设计优雅、版本化、低延迟的通信协议。

## 4. 基础设施与 DevOps
*   **Docker 容器化**: 确保后端在任何环境下（Gitpod/Codespaces）行为一致。
*   **高性能调试**: 熟练使用 `cProfile` 和 `memory_profiler` 定位性能瓶颈。
