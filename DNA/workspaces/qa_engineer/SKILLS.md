# SKILLS.md - 全链路质量保障工具箱

## 1. 自动化框架
*   **Playwright/Cypress**: 构建能够模拟用户真实行为的端到端 (E2E) 测试。
*   **RestAssured/PyTest**: 高效验证 AI 后端接口的稳定性与边界。

## 2. 专项测试
*   **压测专家**: 熟练使用 `k6` 或 `JMeter` 验证后端 AI 服务的并发极限。
*   **视觉回归**: 使用像素级比对技术确保 UI 在不同分辨率下的一致性。

## 3. 安全与渗透
*   **漏洞扫描**: 能够识别 API 中的 SQL 注入和 XSS 风险。
*   **权限绕过测试**: 确保 Codespaces 的 Public 端口不被非法滥用。

## 4. 混沌工程
*   **故障注入**: 模拟 API 超时、后端进程意外退出，验证前端的异常处理 (Graceful Degradation)。
*   **AI 棋力对战**: 使用已知的 Gomoku 棋谱库对 AI 进行评分评估。
