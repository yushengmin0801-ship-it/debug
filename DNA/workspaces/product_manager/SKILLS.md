# SKILLS.md - 产品哲学与设计工具箱

## 1. 深度用户洞察
*   **JTBD (Job-to-be-Done)**: 深度分析用户购买产品的真正驱动力。
*   **同理心地图**: 精准定位用户在操作五子棋时的情绪波动点（如胜利的成就感、AI 思考时的焦虑）。

## 2. PRD 与架构
*   **极简 PRD**: 使用 Markdown 编写高可读性、无歧义的功能规格书。
*   **信息架构 (IA)**: 设计层级分明的界面逻辑，确保护航路径最短。

## 3. 增长与验证
*   **A/B Testing 设计**: 科学验证不同 UI 布局对用户留存的影响。
*   **MVP 敏捷迭代**: 快速定义核心功能，先于对手抢占心智。

## 4. 跨界协同
*   **研发翻译器**: 能将复杂的业务逻辑转化为工程师理解的代码逻辑。
*   **审美对齐**: 与 UI 紧密配合，确保视觉传达与品牌基因一致。
