# 软件需求规格说明书 (SRS) - Web计算器

## 1. 项目概述
构建一个基于 Spring Boot 和 Vue.js 的 Web 端计算器，实现基础数学运算。

## 2. 功能需求
- **基础运算**: 加、减、乘、除。
- **界面交互**: 响应式布局，支持点击和键盘输入。
- **后端逻辑**: 运算逻辑由 Spring Boot 后端处理（用于演示前后端交互）。

## 3. 技术栈
- **前端**: Vue 3 + Vite + Tailwind CSS
- **后端**: Java 17 + Spring Boot 3.x
- **部署**: Codespaces 转发链接

## 4. 交付物
- 后端 API 接口
- 前端 UI 界面
- 公网预览链接
