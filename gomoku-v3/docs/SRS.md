# SRS - Classical Gomoku V3

## 1. Goal
Develop a mobile-friendly, classically styled Gomoku game with a strong Python-based AI.

## 2. Requirements
- **Rules**: 15x15 board, five in a row wins. No forbidden moves (for simplicity).
- **Player**: Human (Black, first) vs. AI (White, second).
- **UI**: Classical Chinese/Japanese aesthetic (Wood textures, ink style).
- **AI**: Heuristic-based evaluation with Minimax/Alpha-Beta search.
- **Mobile**: Responsive design, touch-friendly.

## 3. Tech Stack
- **Backend**: Python (Flask/FastAPI).
- **Frontend**: Vue 3 + Vite.
