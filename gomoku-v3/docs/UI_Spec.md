# UI Design - Classical Gomoku

## 1. Visual Theme
- **Background**: Parchment or Light Wood texture.
- **Board**: Dark ink lines.
- **Stones**: 3D-looking black and white pieces (Black: deep charcoal, White: ivory).

## 2. Layout
- Responsive canvas/grid.
- Side/Bottom panel for status (Turn, Last Move).
- Modal for win/lose announcement.
