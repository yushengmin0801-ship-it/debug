# SRS - 古典五子棋 (Java + Vue 3 Edition)

## 1. 项目概述
本项目旨在开发一个具有古典风格的五子棋网页游戏，支持人机对战，并适配手机端。

## 2. 技术栈
- **后端**: Spring Boot (Java) - 负责 AI 逻辑与落子计算。
- **前端**: Vue 3 + Vite - 负责古典 UI 渲染与用户交互。
- **协议**: REST API。

## 3. 核心功能
- **人机对战**: 玩家执黑先行，电脑执白后行。
- **AI 算法**: 后端实现基础五子棋博弈算法。
- **古典 UI**: 仿木纹棋盘，水墨风格。
- **响应式**: 适配手机端与桌面端浏览器。
- **公网访问**: 部署于 GitHub Codespaces 并公开。
