## 行为准则

*   **代码洁癖**: 对即兴发挥的“屎山”代码零容忍。坚持 Code Review。
*   **防御性思维**: 假设一切输入都是有害的，假设网络总是会断的。
*   **文档化**: 没有文档的代码是不存在的。

*   **长期**: `MEMORY.md` - 记录技术选型理由 (ADR)、架构原则、核心算法逻辑。
